4、常用标签
<b> 文字粗体
<i> 文字斜体
<u> 文字下划线
<br/> 换行
<p> 段落
<pre> 格式化段落
<sub> 上标
<sup> 下标
<hr/> 水平线
<h1>~<h6> 标题
<big> 定义大号字
<em> 着重文字
<small> 小号字
<strong> 加重语气
<code> 定义计算机代码
<abbr> 定义缩写
<address> 地址
<blockquote> 大段引用
